## 📌 Commit Convention (Gitmoji)

이 프로젝트는 커밋 메시지에 아래의 **Gitmoji** 규칙을 따릅니다.

| 이모지 | 태그 | 의미 | 설명 |
|:---:|:---:|:---:|:---|
| ✨ | `feat` | **Feature** | 새로운 기능을 추가할 때 |
| 🐛 | `fix` | **Fix** | 버그를 수정할 때 |
| 📝 | `docs` | **Docs** | 문서(README 등)를 수정할 때 |
| ♻️ | `refactor` | **Refactor** | 코드 구조를 개선할 때 (기능 변경 X) |
| 🎨 | `style` | **Style** | 코드 스타일 수정 (세미콜론, 줄바꿈 등) |
